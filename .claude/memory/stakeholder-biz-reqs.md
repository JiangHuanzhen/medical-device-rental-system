## 招商业务员 — 需求获取记录

### 第1轮（日常工作流程）
**问：** 从接到客户询盘到签合同、后期跟进的完整流程？
**答：** 客户（医院）询盘 → 了解设备需求/租期/预算 → 推荐设备+租金方案（按天/按月/按次）→ 涉及招投标关联标书 → 正式报价（需设底价红线）→ 准备合同（设备清单/租期/维保责任/资质附件）→ 出库配送（送货单+医院验收单）→ 后期跟踪（回款+闲置设备管理）

### 第2轮（系统需求追问）
**问：** 合同管理、回款跟踪、闲置设备的具体需求？
**答：**
- **合同管理：** 预设按天/按月/按次计费模板 → 自动带设备清单+注册证+经营许可证 → 关联招标信息（标书编号/中标金额）→ 一键导出PDF
- **回款跟踪：** 专门"回款看板"按合同列出（已回款/剩余/预计回款日/逾期天数）→ 到期前3天弹窗、逾期每天早上下班推送 → 可发短信给负责人 → 展示客户历史付款记录
- **闲置设备：** 首页卡片/滚动条显示"当前闲置XX台设备" → 点进去看设备状态/存放地点/闲置天数 → 一键生成推荐话术 → 可按地区/设备类型筛选

**Why:** 这些是招商业务员的核心功能需求，直接影响系统租赁订单模块、客户管理模块和首页看板的设计。

**How to apply:** 租赁订单模块需支持灵活的计费方式选择和合同模板；首页看板需有回款提醒和闲置设备概览；客户管理需关联历史付款记录。
