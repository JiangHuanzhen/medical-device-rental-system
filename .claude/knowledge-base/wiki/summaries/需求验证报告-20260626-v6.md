# 需求验证报告

验证日期：20260626
总体结论：**获取类问题**
发现 5 个问题

## 发现清单（教材§7四类交叉比对）
| 类型 | 严重程度 | 涉及章节 | 问题描述 | 修正建议 |
|-----|---------|---------|---------|---------|
| 对话偏差 | 阻塞性 | 3.1 FR-EQP-002 | SRS中紧急出库审批流程的第三步为'科室主任'，但涉众原话（BR-EQP-004）明确要求审批顺序为'库房主管 → 质控部门 → 科室主任'。SRS中'科室主任'角色与需求清单中'使用部门主管'角色不完全对应，存在角色定义偏差。 | 将SRS中'科室主任'修改为'使用部门主管'，或确认'科室主任'是否为'使用部门主管'的别名，并在术语表中明确说明。 |
| 对话偏差 | 重要 | 3.1 FR-EQP-003 | SRS中设备归还检测流程描述为'招商业务员在设备归还界面，点击开始检测按钮'，但涉众原话（BR-EQP-006）明确归还检测应由库房人员执行，且使用与入库检测相同的标准化检查清单。SRS将检测执行者错误地指定为招商业务员。 | 将FR-EQP-003的参与角色从'招商业务员'修改为'库房人员'，并确保检测流程与入库检测一致。 |
| 对话偏差 | 重要 | 3.1 FR-EQP-004 | SRS中FR-EQP-004（配置检测标准）描述为'运维工程师配置检测标准'，但涉众原话（BR-EQP-011）中运维工程师提出的抽检比例配置是针对'设备出库前检测'，而非SRS中描述的'归还检测'。SRS将抽检比例的应用场景错误地限定为归还检测。 | 将FR-EQP-004中'抽检比例'的应用场景扩展至'设备出库前检测'，或明确区分出库检测和归还检测的抽检配置。 |
| 对话偏差 | 建议 | 3.1 FR-ORD-001 | SRS中计费方式列出了6种选项，但涉众原话（BR-ORD-001）中招商业务员明确列出了5种计费方式（按开机次数、按使用时长、按使用量、混合计费、按收入分成），SRS额外增加了'固定周期 + 浮动阶梯'作为第6种。该新增选项未在涉众对话记录中找到明确来源。 | 修正计费方式列表，确保与涉众原话一致，或明确标注'固定周期+浮动阶梯'的来源。 |
| 内部不一致 | 建议 | 3.1 FR-EQP-001 vs 3.1 FR-EQP-002 | SRS中FR-EQP-001（管理设备校准预警）规定'在库、已出租设备到期前7天，系统弹出阻断性弹窗，禁止出库操作'，但FR-EQP-002（执行紧急出库审批）允许对校准过期设备进行紧急出库。两者在'禁止出库'与'允许紧急出库'之间存在逻辑矛盾，未明确紧急出库是否豁免阻断性弹窗。 | 在FR-EQP-001中明确说明'紧急出库审批流程不受此阻断限制'，或在FR-EQP-002中说明紧急出库申请可绕过阻断性弹窗。 |
